The hackathon button navigates to a new screen app/hackathon-login.tsx with an email + password form. On submit it calls supabase.auth.signInWithPassword(). After success, we check hackathon_team_members — if the user is found, we set isHackathon: true and proceed. If not found, show an error: "This account is not registered for a hackathon."

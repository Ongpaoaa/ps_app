# Tech Stack: Passion Seed

## Frontend
- **Framework:** Expo (React Native) - Enables cross-platform mobile development with a single codebase.
- **Language:** TypeScript - Provides strong typing for enhanced developer productivity and code reliability.
- **Navigation:** Expo Router - Simplified, file-based routing for the mobile app.
- **Graphics/Animation:**
    - React Native Reanimated - For high-performance UI animations.
    - Shopify Skia - For advanced 2D graphics and visual effects.

## Backend & Database
- **Backend as a Service (BaaS):** Supabase - Comprehensive backend infrastructure.
- **Database:** PostgreSQL (via Supabase) - Reliable relational database.
- **Server-Side Logic:** Supabase Edge Functions - Scalable, low-latency TypeScript functions for handling complex logic.
- **Authentication:** Supabase Auth - Integrated user authentication and management.

## Project Utilities
- **State Management:** React hooks and component state.
- **Styling:** React Native StyleSheet - Standard styling engine for mobile.
- **Interaction:** Expo Haptics - For tactile user feedback.
- **Engagement:** Expo Notifications - Integrated push notification support.
